# Practice Booking System - Minimal Frontend

This is a minimal frontend for testing the Practice Booking System backend.

## Purpose
- Test backend API endpoints
- Provide basic UI for user testing
- Validate authentication and core workflows

## Not Included (intentionally)
- Advanced styling/design
- Complex animations
- Mobile optimization
- Advanced features

## Quick Start
```bash
cd frontend
npm install
npm start
```

## Testing Credentials
- Use the admin account created during backend initialization
- Check your backend's `.env` file for configured credentials
- API Base: See environment configuration
